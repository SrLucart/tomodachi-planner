# 🏝️ Tomodachi Life: Living the Dream - Island Layout Planner

A basic island layout creator for **Tomodachi Life: Living the Dream**. This project is open-source, so anyone is welcome to grab the code, fork it, and use it as a base to create a more advanced and feature-rich version.

🎮 **[Play Online / Acessar o Planejador](https://SrLucart.github.io/tomodachi-planner)**

## ✨ Features

* **🗺️ Terrain Painting:** Customize the ground with Sea, Sand, Grass, Concrete, Asphalt, and Dirt tiles.
* **🏢 Blueprints & Buildings:** Place Mii Houses, Shops, a Ferris Wheel, Restaurants, and more.
* **📏 Custom Lots:** Need a specific size? Use the Custom Lot tool to define your own width and depth.
* **💾 Save & Load:** Export your island layout as a `.json` file and import it later to continue your work without losing progress.
* **🌍 Bilingual:** Full interface support for English and Portuguese.

## ⌨️ Shortcuts & Controls

To make planning and building faster, the planner includes some handy keyboard shortcuts:

* **`Space` or `R`**: Rotate the selected building blueprint quickly.
* **`Esc`**: Drop the current building or cancel the active tool, returning to the cursor.
* **`Ctrl + Z`**: Undo the last action.
* **`Ctrl + Y`** (or `Ctrl + Shift + Z`): Redo the action.

## 🛠️ How to run locally (For developers)

If you want to use this code as a foundation for your own project, follow these steps:

1. Clone the repository:
`git clone https://github.com/SrLucart/tomodachi-planner.git`

2. Navigate to the project folder:
`cd tomodachi-planner`

3. Install dependencies:
`npm install`

4. Start the development server:
`npm run dev`
